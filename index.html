Poda, derribo y reemplazo de árboles


Igualdad
Es la expresión matemática en la que dos cantidades o expresiones algebráicas tienen el mismo valor.

Ecuación
Es una igualdad en la que hay una o varias cantidades desconocidas (incógnitas) y que es verdadera para determinados valores de las incógnitas.

¿Cuál es la diferencia entre una expresión algebráica y una ecuación?
Una expresión algebráica solo es la unión de variables y números, mientras que una ecuación es una expresión que debe igualarse a una cantidad numérica o a otra expresión.

Continuando con las obras de mantenimiento, se llevó a cabo un censo-diagnóstico del estado físico de los árboles de la Alameda Central estableciendo que, de los 2,285 árboles existentes cierta cantidad están sanos; 265 más que los que están sanos deben derribarse y ser reemplazados y, el triple de los que están en buen estado requieren algún tipo de poda.

1. ¿Es posible formular una ecuación donde se involucren dos cantidades desconocidas, una en relación de otra? (S/N)
 Si

2. ¿Qué tipo de ecuación sería?
 Ecuación lineal

3. Encuentren la ecuación que represente el problema:
 x + (x + 265) + 3x = 2,285

4. Determinen la cantidad de árboles sanos con que cuente la alameda, los árboles para derribo y los que deben ser podados, utiliza la ecuación encontrada.

 x + (x + 265) + 3x = 2,285
 2x + 265 + 3x = 2,285
 5x + 265 = 2,285
 5x = 2,285 - 265
 5x = 2,020
 x = 2,020 / 5
 x = 404


Árboles sanos = 404

Árboles para podar = 1,212

Analicen la siguiente información

 La suma de los costros de pintar las bancas y mantenimiento de luminarias es de $427,440.00. El costo de mantenimiento de las luminarias es de $74,160.00 más de lo que cuesta pintar las bancas.

6. ¿Cuál es el costo de cada una de estas actividades?
 x + (x + 74,160) = 427,440.00
 2x + 74,160 = 427,440
 2x = 427,440 - 74,160
 2x = 353,280
 x = 353,280/
 x = 176,640

Pintar bancas = $176,640
Mantenimiento de luminarias = $250,800

 Dentro del proyecto de remodelación establecen que por cada cuatro bancas se debe colocar un contenedor de basura.

 Si por cada cuatro bancas hay un contenedor de basura, entonces esta relación la podemos representar como:

 contenedor de basura / bancas = 1 / 4

Esta relación también se puede representar mediante una ecuación lineal:

 y = 0.25x

Donde "y" es la cantidad de contenedores de basura y "x" la cantidad de bancas.
